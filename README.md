# pdf_converter

Локальный HTML-конвертер сканированных PDF в Excel (OCR в браузере).

## Запуск

Откройте **`PDF_конвертер_таблиц.html`** в Chrome или Edge.

Файл самодостаточный — папка `js/` для открытия **не нужна**.

## Разработка

Исходники модулей — в папке `js/`. После изменений пересоберите HTML:

```bash
node build.js
```

Шаблон: `PDF_конвертер_таблиц.template.html`  
Резервная копия старой версии: `PDF_конвертер_таблиц.backup.html`
